## 如何使用本程式
* 需要先安裝python
### 步驟一/Step 1
解壓縮資料夾後尋找acoount.txt並將範例改為你的帳號密碼 <br>
Extract the zip file and open account.txt . <br>
Change the examples to your email and password.
```
youremailadrees@gmail.com
yourpassword
```
### 步驟二/Step 2
尋找main.exe點開即可!<br>
Click on main.exe and that's all.
